# rvas
RV automation

A project for motorhomes automation

at the moment is only a skeleton


hardware

raspberry pi zero
esp8266 for sensors

install

on raspberry pi 

install mariadb-server
install nodejs

create db with the script in init_db

sudo node index.js 80 2
